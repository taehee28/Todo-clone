# Todo-clone
To-do App 클론 프로젝트

## 기능
* 제목, 중요도, 내용을 가지는 To-do 항목을 생성, 수정, 삭제 가능
* 저장된 To-do 항목을 검색 가능
* To-do 목록을 중요도 기준으로 정렬 가능
* 저장된 To-do 항목 전체를 삭제 가능
* 개별 To-do 항목 삭제 시 Undo 가능 
* To-do 목록에서 개별 항목을 옆으로 Swipe 하여 삭제 가능

## 구현에 사용한 요소
* Kotlin
* Compose
* Navigation
* Room
* Hilt
* DataStore
